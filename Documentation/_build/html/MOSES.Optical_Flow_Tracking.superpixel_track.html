MOSES.Optical_Flow_Tracking.superpixel_track.compute_vid_opt_flow(vidstack, params, mask=None, adaptive=False, ksize=3)[source]¶

Given a grayscale movie as a numpy vidstack, with the specified optical farneback params as params. Returns the stacked optical flow (frame by frame) for the entire video with optional masking.

Set mask=None, adaptive=False for no masking.

Parameters:
vidstack
: numpy array

n_frames x n_rows x n_cols numpy array

params
: python dict

OpenCV opticalflowfarneback parameters (python dict) c.f. http://docs.opencv.org/2.4/modules/video/doc/motion_analysis_and_object_tracking.html

mask
: numpy bool array

optional n_rows x n_cols (single frame) boolean mask to only compute the optical flow in selected regions.

adaptive
: bool

if mask is not given then specify if mask will be determined adaptively frame to frame using Otsu thresholding of pixel intensity.

ksize
: int

size of disk kernel to dilate the mask if adaptive is set to True.

Output:
——-
flow :

(n_frames-1, n_rows x n_cols x 2) numpy array giving the optical flow
MOSES.Optical_Flow_Tracking.superpixel_track.count_point_density(poslist, spixelsmap, min_density=1)[source]¶

Counts the number of instances coordinates in poslist falls into each superpixel region specified by spixelsmap and determine superpixel regions whose region has less than the minimum required coordinate density.

Parameters:
poslist
: numpy array

(n_points x 2) numpy array with channels in image coordinate (y,x) order.

spixelsmap
: numpy array

the number of regular region of interst partitions (ROI) of the image. Each unique region is denoted with an integer from 0 to N-1 where N is the total number of regions.

min_density
: int

the minimum required number of points a region should contain.
Returns:
pos_density
: numpy array

vector with number of points within each initial superpixel region.

no_dense_regs
: numpy array (vector of int)

vector of each initial superpixel region which do not have the mindensity number of points within.
